# [[문제]](https://www.acmicpc.net/problem/1003)

### DP를 활용했다.

0이 출력되는 횟수와 1이 출력되는 횟수를 나누어서 출력하는게 기본 출력 조건이다.

피보나치 함수가 현재 보고 있는 시점의 전전과 전번째 순서의 수를 더한 값이므로

그때의 전전과 전번째 0이 출력되는 횟수를 더한 값이 현재의 0의 개수가 되고

마찬가지로 1의 개수도 구할 수 있다.
